# 📚 CodeAlpha - Simple E-commerce Store

## 📖 Project Overview
This is a Simple E-commerce Store developed as part of the CodeAlpha Full Stack Development Internship.

The project allows users to browse books, register/login, add products to a shopping cart, view product details, and place orders.

## 🚀 Features

- User Registration
- User Login
- Product Listing
- Product Details Page
- Shopping Cart
- Order Processing
- Responsive UI
- Express.js Backend

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## 📂 Project Structure

```
CodeAlpha_EcommerceStore
│
├── frontend
├── backend
├── package.json
└── README.md
```

## ▶️ How to Run

1. Open Terminal
2. Go to backend folder

```
cd backend
```

3. Install dependencies

```
npm install
```

4. Start the server

```
node server.js
```

5. Open the frontend in your browser.
